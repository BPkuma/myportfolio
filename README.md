# 注意！
ローカル環境でテストする際にはターミナルで以下を実行しないとバグります<br>
1  `sail up -d`<br>
2  `sail npm run dev`<br>
<br>
2についてはバックグランドで実行できないっぽいのでターミナルを占有します。コマンド操作するには新しくターミナルを立ち上げないといけないようです。<br>
